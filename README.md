# Number Guessing Game

Welcome to the Number Guessing Game! This is a simple Python project where you can test your guessing skills by trying to guess a randomly generated number between 1 and 100.

## How to Play

1. Clone the repository to your local machine.
2. Open your terminal or command prompt.
3. Navigate to the project directory.
4. Run the game by executing the following command:

    ```bash
    python Guess_the_number.py
    ```

5. The program will generate a random number between 1 and 100.
6. You'll be prompted to guess the number.
7. Enter your guess and hit Enter.
8. If your guess is incorrect, the program will give you a hint and allow you to guess again.
9. Continue guessing until you find the correct number.
10. The program will provide feedback on the number of attempts it took you to guess the correct number.

Have fun and enjoy the game!
